# WholeSalePosWithMavenSpringJPA

### Installation

1. Initialize git repository
```sh
git init
```
2. Clone the repo
```sh
git clone https://github.com/AnujaKoralage/WholeSaleMavenSpringJPA.git
```
3. Import project via Intellij Idea
